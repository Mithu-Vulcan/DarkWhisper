# Dark Whisper - Flask Chat App

Dark Whisper is a real-time chat application built using Flask. It is designed for seamless and efficient communication with a simple and intuitive interface.

## Features
- Real-time messaging
- User authentication (login/register)
- Responsive UI
- Flask backend
- WebSocket integration (to be implemented)
- Lightweight and efficient

## Installation

### Prerequisites
Ensure you have the following installed:
- Python 3.12
- Flask
- Flask-SocketIO (if WebSockets are implemented)

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/Mithu-Vulcan/DarkWhisper/.git
   cd dark-whisper
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
4. Open your browser and go to:
   ```
   http://127.0.0.1:5000
   ```
